Architecture

Three-Tier Routing Pipeline

1 Deterministic 15 regex/rule solvers catch pure math, date math, unit conversions, sentiment, counting, word problems — 0 tokens
2 Weighted Classifier Regex scoring (weighted per-pattern) + fastembed cosine-similarity fallback; maps to 8 categories (code, math, logic, NER, sentiment, summarization, factual)
3 Local LLM Ensemble Qwen2.5-0.5B (GGUF, CPU) runs 3 temperatures (0.0, 0.3, 0.7); consensus ≥ 67% accepted — 0 API tokens
4 Local LLM Single Falls through from ensemble; single pass (temp=0.1) with same local model — still 0 API tokens
5 Fireworks API Per-category model tier, system prompt, max_tokens, and temperature from prompts.py, with retry on failure

Steps 1-4 cost zero Fireworks tokens. The API is called only when all local tiers fail.

Key Technique

Three-Tier Local Escape Hatch

Before any API call, three local tiers attempt the task in sequence. Resolution at any tier means zero Fireworks tokens.

1 Deterministic 15 regex/rule solvers — 0 tokens, perfect accuracy, no model needed
2 Local LLM Ensemble Qwen2.5-0.5B @ 3 temps — consensus gate (≥ 67%) rejects low-confidence outputs
3 Local LLM Single Same model, single pass (temp 0.1) — catches what the ensemble couldn't agree on

Only if all three local tiers fail does the task escalate to the Fireworks API — minimizing token spend while maximizing solve rate.
