TokenSage

Hybrid Token-Efficient Routing Agent

Minimize Fireworks AI token spend by solving tasks deterministically where possible and using optimized per-category API calls only when needed.

Problem

The Token Efficiency Challenge

Deterministic solves cost nothing

Regex and rule-based solvers handle pure math, dates, sentiment, extraction, and more — zero tokens, perfect accuracy.

Every Fireworks token costs you

The leaderboard ranks by token spend. The winner uses the fewest Fireworks tokens while meeting the accuracy gate.

Minimize API calls

No GPU or local model is guaranteed. The router must solve what it can with pure logic, and send only the rest to Fireworks with optimized budgets.

Architecture

Routing Pipeline

1 Deterministic 11 regex/rule solvers catch pure math, date math, unit conversions, sentiment, counting, word problems — 0 tokens
2 Classifier Regex-based task categorizer maps prompts to 8 types (math, sentiment, NER, factual, summarization, etc.)
3 Fireworks API Per-category model tier, system prompt (4-8 words), max_tokens (50-800), and temperature, with auto-retry on failure

Deterministic solves (step 1) cost zero Fireworks tokens — they run before any API call is made.

Key Technique

Deterministic Pre-Solving

Before any API call, 11 lightweight rule-based solvers attempt the task. If one matches, the answer is returned immediately — zero tokens, perfect accuracy.

What gets solved

Arithmetic, unit conversion, date math

Sentiment analysis (lexicon-based)

JSON parsing, email/URL extraction

Counting, true/false, percentage, word problems

Why it works

Pure regex and Python eval — no model needed, 0 tokens

100% deterministic: same input always gives same answer

Covers the long tail of simple tasks that would waste API tokens

Anything the deterministic solvers don't catch falls through to the classifier and then the Fireworks API.
